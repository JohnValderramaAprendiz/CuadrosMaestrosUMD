Instructivo de Uso de Quantum

Bienvenido al instructivo de Quantum. Sigue estos sencillos pasos para generar tus Cuadros Maestros de forma eficiente.

Pasos para Generar un Reporte:

  1. Navega a Cuadros Maestros: Haz clic en "Cuadros Maestros" en el menú de navegación para ver todas las plantillas disponibles.
  2. Busca tu Plantilla: Utiliza el campo de búsqueda para encontrar rápidamente la plantilla que necesitas (ej. "Prácticas" o "Becas").
  3. Selecciona y Genera: Haz clic en el botón "Generar" de la tarjeta de la plantilla deseada.
  4. Ingresa el Código: En la ventana emergente, introduce el Código de Programa (SNIES o el código específico que se te solicite) para tu programa académico.
  5. Descarga el Reporte: Haz clic en "Generar y Descargar Reporte". El sistema procesará los datos y el archivo Excel se descargará automáticamente en tu dispositivo.

Consideraciones Importantes:

  • Asegúrate de que el código de programa ingresado sea correcto para obtener el reporte adecuado.
  • Si tienes problemas para generar o descargar un reporte, verifica tu conexión a internet o contacta al equipo de soporte.
  • Para más información sobre las plantillas o la estructura de los datos, visita la sección de Repositorio.
